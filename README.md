 SQL - Restaurante

Projeto de banco de dados em SQL para um restaurante fictício.

Etapas realizadas

* Criação do banco de dados `restaurante`
* Criação das tabelas:

  * funcionarios
  * clientes
  * produtos
  * pedidos
  * info_produtos
* Inserção de dados fictícios
* Consulta dos registros com SELECT
* Atualização de funcionário por CPF
* Atualização de status dos pedidos
* Remoção de funcionário por CPF

Arquivos

* `definicao.sql`
* `manipulacao.sql`

Ferramentas utilizadas

* MySQL Workbench
* SQL
* GitHub
